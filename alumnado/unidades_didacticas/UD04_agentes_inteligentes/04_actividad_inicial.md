# UD04 - Actividad Inicial: Mapeo de Intenciones a Funciones

## Objetivos
Analizar un conjunto de mensajes de usuario y diseñar la firma de la función Python que debería resolver cada petición de forma automatizada.

## Tareas
1. Para el mensaje *"¿Cuántas unidades quedan de la laptop X?"*, definir el nombre de la función, sus argumentos y tipos.
2. Identificar riesgos de seguridad si el modelo generase una llamada arbitraria.
