# UD05 - Actividad Inicial: Exploración de Swagger UI en FastAPI

## Objetivos
Arrancar un servidor de prueba con Uvicorn e interactuar con la interfaz Swagger UI (`/docs`).

## Tareas
1. Instalar FastAPI con `uv add fastapi uvicorn`.
2. Arrancar el servidor en modo recarga: `uv run uvicorn main:app --reload`.
3. Navegar a `http://localhost:8000/docs` y probar el endpoint de salud `/health`.
