# UD05 - Reto de Ampliación: Stack Multi-Servicio en Docker Compose

## Reto
Crear el archivo `compose.yaml` que orqueste en una misma red bridge el servicio `api` (construido desde el Dockerfile local) y el servicio `ollama`, verificando que la API pueda comunicarse internamente con Ollama por su nombre de servicio `http://ollama:11434`.
